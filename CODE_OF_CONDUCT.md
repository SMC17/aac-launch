# Contributor Covenant Code of Conduct

This project adopts the [Contributor Covenant v2.1][1]. By participating,
you agree to abide by its standards.

[1]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Enforcement

Reports go to the maintainer via the channel listed in `SECURITY.md`.

## Scope

This code of conduct applies in all project-related spaces — GitHub
issues, PRs, discussions, and any community channel that adopts this
project's name.
